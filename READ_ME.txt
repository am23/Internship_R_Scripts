This folder contains all the scripts,datasets and image-results for prediction, forecasting and comparison that have been applied till now in R and matlab.

1)Airpassengers_R contains all the models with datasets and results to show prediction airpassenger using R.

2)Airpassengers_Matlab contains all the models with datasets and results to show prediction airpassenger using matlab.

3)Melbourne_Sydney contains the comparison results for the 'melsyd' dataset

4)neural_matlab contains k-ahead prediction done in matlab

5)Neural Networks using neuralnet package

6)salestimeseries dataset using neural networks to predict output

7) predicting sine curve using neuralnetwork
